# 星際邊緣 · 第八章

---

**星際公民**

李明遠站在遺跡的六邊形平台上，四周站著七個國家的技術代表。星環的藍光在他的腳下緩緩流動，像一片液态的星空。

「你真的願意？」林奕如問。

「嗯。」

「你知不知道他們說的是什麼？『星際公民』——這意味著你可能再也回不到地球了。」

「我知道。」

林奕如望著他，眼神複雜。「為什麼？」

「因為我想知道答案，」李明遠說，「UFO 為什麼選擇這座島？星環裡面隱藏什麼？還有……那些文字說『當文明學會共存，才算真正成年』——我想看看，成年後的宇宙是什麼樣子。」

---

**【最後的傳送**

平台的中央開始浮現光點。不是蓝光，而是金色的。

```
傳送協議啟動。
目標：天狼星星系，星際邊緣哨站。
```

「天狼星……」一位技術代表喃喃自語，「距離地球八點六光年。這是怎麼做到的？」

「折叠空间，」李明遠說，「UFO 用的就是這種技術。」

金色光芒越來越高，最終淹沒了整个平台。林奕如下意识地想去拉李明遠的手，但光芒太盛，她被迫闭上眼睛。

當她再次睜開眼時，平台上已經空了。

只剩那句話，刻在六邊形晶體的表面：

```
旅程才剛開始。
```
