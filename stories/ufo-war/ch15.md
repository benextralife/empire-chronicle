# 星際邊緣 · 第十五章

---

**星際邊緣 · 第二季：紙上的星光**

林曉薇收到了一個包裹。

沒有寄件人，沒有地址，只有一個六邊形的圖案。她拆開包裹，發現裡面是一疊紙——纸的質感很奇怪，像是某种植物的纖維，卻又比紙更堅韌。

紙上面寫滿了字。是中文。

```
親愛的林曉薇：
當你讀到這句話的時候，我已經離開了星際邊緣哨站，前往更遠的星系。
```

紙的末尾，是一幅畫。畫的是一個人站在星空下，抬頭望著宇宙的盡頭。畫的角落，有一行小字：

```
因為你，我學會了孤独。
```

林曉薇望著窗外的大海，淚水終於落了下來。
