# 星際邊緣 · 第十四章

---

**基地島 · 第一百天**

外星學者們已經離開了。但他們留下來的東西，改變了整座島——甚至是整顆星球。

「能源問題解决了，」林曉薇在新聞發布會上說，「借助星際科技，我們現在可以免費獲得清洁能源。氣候變化、糧食短缺、貧富差距——這些困擾人類數千年的問題，終於有了答案。」

記者問道：「這是不是意味著，地球文明已经進入了『星際時代』？」

「不是，」林曉薇笑著搖頭，「這才剛剛開始。」

---

**【新的開始】

發布會結束後，林曉婉獨自來到遺跡洞口。那裡已經變成了紀念公園，遊客們在洞口拍照，孩子们在草地上奔跑。

她把手放在六邊形晶體的表面上，那熟悉的蓝光再次亮起。

「你還好嗎？」她低聲問。

```
我很好。而且，我在學習寫詩。
```

林曉薇笑了。「寫給誰？」

```
寫給所有曾經害怕我的你們。
```

**【第十四章 完 · 第一季正式完結】**
