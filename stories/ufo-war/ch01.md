# 星際邊緣 · 第一章

---

公元 2097 年，地球不再是孤立無援的藍色星球。

那天晚上，李明遠加班到十一點，開著那輛老舊的油電混合車，獨自駛在環島中線省道。路燈一盞一盞，像默哀的螢火蟲。他剛處理完一場全球晶片供應鏈的衛星數據異常，腦袋裡還是那個反覆出現的錯誤碼：`ERR_SIGNAL_LOST`。

天空沒有任何預兆。

先是無線電靜音——FM 預設電台的訊號像被掐住喉嚨，只剩下沙沙的白噪音。車載導航螢幕閃爍了一下，跳出一個他從未見過的圖形：一個完美的圓環，由六段發光的弧線組成，在螢幕中央緩慢旋轉。

「這是什麼鬼東西？」

他伸手要觸碰螢幕，突然，整條省道的路燈同時熄滅。

不是閃爍，不是斷電，而是整整齊齊地、同步地暗了下去。懾人的沉默籠罩了平原。李明遠減速，從車窗望出去，遠方的山脊線上，一個銀白色的碟形物體正緩緩下降，沒有聲音，沒有引擎噴火，只有空氣被擠開時那種低頻的嗡鳴。

它的速度很慢，卻完全違背物理定律——速度感讓他的視網膜感到刺痛，但駛音卻異常低沉，像一個巨人在耳邊呼吸。

李明遠沒有慌張。他是這座島上國防科技研究院的資深工程師，見過模擬場景，受過緊急應變訓練。第一時間，他撥打了緊急通報號碼。

但電話沒有訊號。不僅是他的手機，就連他隨身攜帶的軍事頻道對講機也變成了啞巴。

碟形物體懸停在離地約三百公尺的空中，底部打開一個圓形的口，一道淡藍色的光柱投射下來，像探照燈一樣掃過麥田。光柱所到之處，作物沒有燃燒，也沒有枯萎，而是變得透明，像被某種高頻能量共振到了分子層級。

「它在採樣。」

李明遠腦中閃過這個念頭。他悄悄按下車上的緊急按鈕——那不是普通的警報按鈕，而是研究院最近測試的「異常事件快速通報協定」。按鈕連接著一個隱蔽的衛星通道，專為這種「無法解釋」的情況設計。

螢幕上的圓環圖形再次閃爍。這一次，出現了文字：

```
檢測到地球智慧體。
評估中：威脅等級 — 待定。
倒數計時啟動。
```

倒數計時？什麼倒數？

李明遠還沒反應過來，那道光柱突然轉向，直奔他的車而來。

他猛踩油門，老舊的 hybrid 引擎發出抗議的嘶吼，車輛彈射出去。光柱擦過後保險桿，瞬間讓後半段的金屬結構變成了半透明，像被強酸腐蝕後的骨架。後視鏡熔化了，滴落在柏油路上，發出滋滋聲。

他駛進了一座小型停車場，躲在混凝土建築之間。碟形物體似乎失去了目標，緩緩調整方向。

就在這時，天空出現了第二個光點。

不是碟形物體，而是一枚拖著火焰的導彈——從這座島東海岸的海上發射，穿越雲層，精準地朝不明飛行物沖去。是國軍的反應夠快，還是早就有所準備？

爆炸的光照亮了半個夜空。

但那艘 UFO 並沒有墜落。它在爆炸發生的瞬間分裂成數十個小型碟狀機群，像蒲公英的種子一樣散開。每一艘小碟都發射出乾擾脈衝，導彈在半空中偏離軌道，墜入海裡。

更可怕的是，那些小碟開始反擊。一道看不見的電磁脈衝掃過停車場地面的車輛，所有電子設備同時失靈。李明遠的車熄火了，手機變成了磚塊，就連他手腕上的機械——是的，機械——手也停了，指針垂了下來。

他聽見自己的心跳。還有，遠處傳來的金屬扭曲聲。

那不是地球的聲音。

---

**同一時間，太平洋深處**

一艘隸屬北方聯盟的潛艦靜默地潛航在基地島外海五十海浬處。指揮艙內，艦長盯著聲納螢幕上那些不規則的幾何圖形，臉色鐵青。

「報告目標數量。」

「至少四十個，艦長。它們的材質不吸聲波，標準魚雷鎖定不了。」

艦長沉默了。這座島上的晶片工廠，供應了全球 70% 最先進的 3 奈米與 2 奈米處理器。島上那個疑似外星遺跡的訊號，已經讓三個超級大國暗中角力了三年。

現在，它們來了。

「上級怎麼說？」

「奪島行動凍結，」通訊官的聲音有些顫抖，「上級說——先讓他們狗咬狗。我們最後進場。」

艦長望著漆黑的導航螢幕，知道這句話的意思是：島上的居民、各國的間諜、還有天上那四十艘不屬於地球的飛行器，統統都是棋子。

誰贏了最後那一役，誰就握住了這顆星球的命脈。

---

國防部地下指揮中心，緊急應變會議進行到第三刻鐘。

投影幕上播放著衛星拍到的畫面，三十多個軍方高官臉色凝重。

「不是單一目標，」李將軍指著螢幕，声音堅硬如鐵，「它們是蜂群戰術。我們對抗的不是來訪者，是入侵者。而且，它們在採樣我們的科技。」

「有沒有交渉的可能？」

李將軍搖了搖頭。「第一道光柱就是答案。它們不打算談判。」

幕外，一名年輕的技术官低聲對同事說：「如果它們能分解我們的電子裝置，說不定也能分解我們的……生物結構。」

對方沒有回答。

李明遠站在角落，聽著這些對話，腦中浮現那個倒數計時。他從懷中掏出那支在爆炸中奇跡幸存的行動電源——外殼已經龜裂，但還能運作。他接上早已關機的平板電腦，重啟。

螢幕亮起的瞬間，所有人都轉過頭来。

那個圓環圖形又回來了。這一次，伴隨著一行不斷變化的數字：

```
T-MINUS: 72:00:00
```

**七十二小時後，發生了什麼？**

李明遠深吸了一口氣，知道從今晚起，他的人生只剩一個任務：在倒數歸零之前，找到方法，讓人類贏得這場原本不對稱的戰爭。
