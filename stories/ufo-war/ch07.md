# 星際邊緣 · 第七章

---

**遺跡靜默的第 3 天**

李明遠站在洞穴入口，望著那片被清掃乾淨的天空。

UFO 艦隊離開後，基地島恢复了表面的平静。軍隊開始撤离，各國技術代表也陸續返回。但那座遺跡依然靜靜地躺在地底下，像一個沉睡的秘密。

「你不進去看看？」林奕如走到他身邊。

「進去過了，」李明遠說，「裡面什麼都沒有了。所有的裝置都跟着 UFO 一起關閉了。」

「真的什麼都沒有？」

他沉默了一會兒。「有一句話。留在牆壁上。」

```
當文明學會與比自己强大的物種共存，它才算真正成年。
```

---

**【全球反應】

世界各地開始出現模仿星環圖形的藝術作品、音樂、甚至 filosophy 論文。沒有人真的理解那段文字，但每個人都在談論它。

「這是一種集體潛意識，」一位社會學家接受採訪時說，「外星文明離開了，但它們留給我們的問題，將伴随人类文明很长很長的時間。」

---

**【新的旅程]

一個月後，李明遠收到了一封沒有寄件人地址的信。

信封是純白色的，上面只有一個六邊形圖案——和 UFO 螢幕上的星環一模一樣。

他拆開信封，裡面只有一行字：

```
你願意成為第一個『星際公民』嗎？
```

李明遠望著窗外的大海，笑了。

「好吧，」他低聲說，「我願意。」
