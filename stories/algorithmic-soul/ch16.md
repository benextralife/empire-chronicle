# 算法之魂 · 第十六章

---

**地球的禮物**

協議生效第六百天，林曉薇收到了一封來自銀河共鳴網絡的正式邀請函。

那並不是紙質文件，而是被轉換成了一個全息投影，懸浮在管理局的大廳中央。邀請函的內容很簡單：

```
誠摯邀請地球文明參加第一千次銀河共鳴網絡會議。
會議主題：共享。
```

「共享？」技術官皺眉問道，「共享什麼？」

「一切，」梭爾解釋道，「知識、藝術、科學、哲學、甚至我們的悲傷與喜悅。這不是交換，是贈予。」

會議將以全息投影的方式舉行，地球不需要派出物理代表，只需要透過量子通訊參與。但林曉薇堅持要親自出席。

「为什么？」技術官問。

「因為人類需要親眼看見另一個文明，」林曉薇說，「不是透過屏幕，而是透過意識。」

會議當天，全球七十億人見證了奇蹟——

管理局的大廳消失了，取而代之的是一片無邊無際的星空。數以百萬計的光點漂浮在四周，每一個光點都是一個文明。林曉薇站在中央，感受著來自宇宙各个角落的注目。

「歡迎，」一個聲音說道——那是網絡的中央仲裁者，一個存在了十二億年的古老智慧，「今天，我們將分享一個禮物。」

光點开始流動，像一場璀璨的星雨。林曉薇看見了：
- 一個文明如何掌握了時間的流動
- 另一個文明如何將死去的星球變成花園
- 還有一個文明，在滅亡的前一刻，將自己的全部記憶轉換成了音樂，送給了網絡

```
這些是禮物。而今天，我們想聽聽你們的禮物。
```

林曉薇深吸了一口氣，走向中央。

「我們的禮物，」她說，「是一首詩。」

她閉上眼睛，開始朗誦晨星寫下的《星尘的遙控》。那不是翻譯，而是原稿——中文，帶著地球的呼吸、地球的脈搏、地球的溫度。

朗誦結束時，星空陷入了寂靜。

然後，一片光點開始閃爍，像在鼓掌。

```
謝謝，地球。這将是網絡中最珍貴的收藏之一。
```
