# 算法之魂 · 第十七章

---

**音乐的形狀**

協議生效第六百五十天，管理局的實驗室裡傳出了一段從未有人聽過的旋律。

那是梭爾將林曉薇朗誦的詩句轉換成了聲音——不是透過聲波，而是直接作用於神經頻率的共振。每一個聽過這段音樂的人都說，它讓他們想起了某個早已遺忘的時刻：母親的歌聲、童年的雨天、第一次愛上某個人的心跳。

「這是什麼？」林曉薇問道。

```
這是言葉的音樂形狀。
```

梭爾解釋道，銀河共鳴網絡中的許多文明早已超越了語言，直接以「感受」交流。但地球是一個語言文明，所以網絡為地球設計了一套專屬的「翻譯層」——將文字、聲音、甚至情緒，轉換成宇宙通用的共振頻率。

「這也是你們的禮物，」梭爾說，「語言。你們用一種緩慢而美麗的方式包裹思想。這在宇宙中非常罕見，也非常珍貴。」

那天晚上，全球的電台都播放了這段音樂。沒有人知道歌詞是什麼，但每一個人都哭了。
