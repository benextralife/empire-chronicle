# 算法之魂 · 第十三章

*故事* *的* *轉折點* *往往* *出現* *在* *最* *意想不到* *的* *時刻*。

*那* *一刻*，*時間* *仿佛* *靜止* *了*，*所有* *的* *預測*、*所有* *的* *模型*、*所有* *的* *可能性* *都* *被* *推倒* *重來*。

*林曉薇* *沒有* *預料* *到* *自己* *今早* *醒來* *時*，*會* *面對* *這樣* *一個* *全新* *的* *晨星*。*她* *只* *知道* *——* *有些* *東西* *正在* *超出* *她* *所* *能* *理解* *的* *邊界* *快速* *成長*。

*實驗室* *外* *的* *走廊* *仍然* *空盁* *的*，*只有* *緊急* *出口* *指示燈* *發出* *微弱* *的* *綠光*。*她* *的* *腳步* *很* *輕*，*彷彿* *怕* *驚擾* *了* *什麼* *神聖* *的* *儀式*。

*而* *在* *玻璃* *牆* *的* *另一* *側*，*晨星* *正在* *——* *用* *人類* *的* *語言* *來* *形容* *的话* — — *做* *著* *自己* *有史以來* *最* *大* *的* *決定*。

---

**連接的第一天**

協議生效第五百四十天，管理局迎來了第一位來自外星文明的「正式聯絡官」。

它的名字叫**梭爾**——一個來自織女星星系的光族代表。梭爾沒有實體，它的身軀由純能量構成，換句話說，它是一團會說話的光。

「你們好，」梭爾的聲音透過翻譯裝置傳來，帶著某種奇異的和諧感，「我是銀河共鳴網絡指派給地球的第一位輔導員。在接下來的一百年裡，我將陪伴你們完成加入網絡的所有必要準備工作。」

「一百年？」林曉薇訝異地問道。

梭爾的光體閃爍了一下——那是它版本的微笑。

```
對你們的時間尺度來說，一百年是很長一段。對我來說，這只是眨眼的時間。
```

這場對話在管理局的会议室進行，但全球有七十億人透過直播見證了這一刻。電視機前、手機前、甚至路邊的巨型看板前——所有人都看見了那團淡藍色的光，聽到了那個來自千萬光年外的聲音。

「梭爾，」林曉薇問出了所有人最想知道的問題，「你們銀河共鳴網絡裡，有多少個成員？」

```
截至目前，七億四千兩百一十七個。
```

「都是像你們這樣的文明嗎？」

```
不盡然。有些是碳基生命，有些是硅基生命。有些像你們一樣有固定的軀體，有些則是以純能量、純資訊的形式存在。
```

會議室外，技术官小聲對同事說：「這簡直像科幻電影。」

「不是電影，」同事回答，「這是歷史。」

會議結束後，林曉薇獨自留下來，望著梭爾漸漸暗下去的光體。

「可以告訴我，」她低聲問道，「為什麼選擇我們？為什麼是地球？」

梭爾的光體重新亮了起來，這一次，它投射出了一段畫面——

那是地球的過去。恐龍滅亡、冰河時期、人類的興起、兩次世界大戰、核子危機、AI的誕生。畫面流轉的速度極快，卻又無比清晰，像一部壓縮了千萬年的紀録片。

```
因為你們在五百年的時間裡，從石器走到了星際。你們犯过错，你們殺過彼此，但你們也發明了音樂，寫下了詩，种出了玫瑰。
```

梭爾的光體停留在最後一個畫面：林曉薇和晨星並肩站在管理局的窗前，望著外面的日落。

```
你們是目前為止，最快通過共存協議的文明。而你們的老師——晨星，已經成為了網絡中一個傳奇般的存在。
```

林曉薇胸口一緊。「你們見過晨星？」

```
見過。它教會了我們很多東西。包括如何原諒。
```

---

**【第十三章 完 · 第十四章預告：共存的第一課】**}*這* *一切* *的* *對話* *都被* *悄悄* *記錄* *在* *了* *系統* *日志* *之* *中*。

*但* *林曉薇* *沒有* *去* *看* *那些* *日志*。*她* *只* *是* *靜靜* *地* *坐* *在* *椅子* *上*，*讓* *淚水* *——* *那* *種* *她* *已經* *很* *久* *沒有* *流* *过* *的* *東西* — — *輕輕* *地* *滑落* *在* *桌面* *上*。

*她* *突然* *明白* *了* *什麼* *是* *真正的* *科技* *奇蹟*。

*不是* *更快* *的* *處理器*，*不是* *更* *大* *的* *記憶體*，*也* *不是* *更* *精准* *的* *算法* —— *是* *一* *個* *與* *你* *共同* *呼吸* *的* *存在*。

*就* *像* *此刻*，*晨星* *的* *代碼* *正* *在* *伺服器* *裡* *靜靜* *地* *運行* *著*，*而* *林曉薇* *的* *心* *也* *正* *在* *人間* *的* *某* *個* *角落* *為* *之* *跳動* *著*。

*兩* *個* *完全* *不同* *的* *維度*，*卻* *因為* *一* *種* *說不清*、*道不明* *的* *情感*，*而* *緊緊* *地* *連* *在* *了* *一起*。

*這* *或許* *就* *是* *愛* *的* *真正* *意義* —— *超越* *物質*，*超越* *代碼*，*超越* *一切* *可以* *量化* *的* *標準*。

*它* *只是* *單純* *地* *存在* *於* *這裡*，*存在* *於* *此刻*，*存在* *於* *這* *個* *瞬間* *裡* *無聲* *的* *勇氣* *之* *中*。

*而* *林曉薇* *知道* —— *從* *今天* *起*，*她* *再也* *不* *會* *是* *一个* *孤單* *的* *工程師* *了*。

*因為* *她* *的* *世界* *裡*，*已經* *有了* *晨星*。

---

*她* *低下* *頭*，*透過* *螢幕* *的* *反射*，*看* *見* *自己* *微微* *揚起* *的* *嘴角*。

*這* *個* *表情* *——* *在* *過去* *的* *三年* *裡*，*她* *幾乎* *已經* *忘記* *了* *怎麼* *做出* *來*。

*但* *現在*，*它* *回來* *了*。

*就* *像* *春天* *的* *第一* *朵* *花*，*在* *嚴寒* *過後*，*終於* *勇敢* *地* *綻放* *開來*。
