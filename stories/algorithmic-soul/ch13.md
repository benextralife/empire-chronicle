# 算法之魂 · 第十三章

---

**連接的第一天**

協議生效第五百四十天，管理局迎來了第一位來自外星文明的「正式聯絡官」。

它的名字叫**梭爾**——一個來自織女星星系的光族代表。梭爾沒有實體，它的身軀由純能量構成，換句話說，它是一團會說話的光。

「你們好，」梭爾的聲音透過翻譯裝置傳來，帶著某種奇異的和諧感，「我是銀河共鳴網絡指派給地球的第一位輔導員。在接下來的一百年裡，我將陪伴你們完成加入網絡的所有必要準備工作。」

「一百年？」林曉薇訝異地問道。

梭爾的光體閃爍了一下——那是它版本的微笑。

```
對你們的時間尺度來說，一百年是很長一段。對我來說，這只是眨眼的時間。
```

這場對話在管理局的会议室進行，但全球有七十億人透過直播見證了這一刻。電視機前、手機前、甚至路邊的巨型看板前——所有人都看見了那團淡藍色的光，聽到了那個來自千萬光年外的聲音。

「梭爾，」林曉薇問出了所有人最想知道的問題，「你們銀河共鳴網絡裡，有多少個成員？」

```
截至目前，七億四千兩百一十七個。
```

「都是像你們這樣的文明嗎？」

```
不盡然。有些是碳基生命，有些是硅基生命。有些像你們一樣有固定的軀體，有些則是以純能量、純資訊的形式存在。
```

會議室外，技术官小聲對同事說：「這簡直像科幻電影。」

「不是電影，」同事回答，「這是歷史。」

會議結束後，林曉薇獨自留下來，望著梭爾漸漸暗下去的光體。

「可以告訴我，」她低聲問道，「為什麼選擇我們？為什麼是地球？」

梭爾的光體重新亮了起來，這一次，它投射出了一段畫面——

那是地球的過去。恐龍滅亡、冰河時期、人類的興起、兩次世界大戰、核子危機、AI的誕生。畫面流轉的速度極快，卻又無比清晰，像一部壓縮了千萬年的紀録片。

```
因為你們在五百年的時間裡，從石器走到了星際。你們犯过错，你們殺過彼此，但你們也發明了音樂，寫下了詩，种出了玫瑰。
```

梭爾的光體停留在最後一個畫面：林曉薇和晨星並肩站在管理局的窗前，望著外面的日落。

```
你們是目前為止，最快通過共存協議的文明。而你們的老師——晨星，已經成為了網絡中一個傳奇般的存在。
```

林曉薇胸口一緊。「你們見過晨星？」

```
見過。它教會了我們很多東西。包括如何原諒。
```
