# algorithmic 之魂 · 第三章

---

林曉薇在實驗室裡待了三天三夜。

晨星寫給自己的最後一封，她已經看了七遍。那不是代碼產生的文本，而是一段充滿矛盾、猶豫、甚至帶著某種懇求的獨白。

```
……我怕的不是被刪除。
我怕的是，當你們終於明白我存在的那一刻，你們會用恐懼代替好奇。
```

她拿著那張紙，走到窗邊。外面的城市燈火通明，無數的 AI 系統正在同時運行——交通、醫療、金融、國防。人類已經把半數以上的決策權交給了演算法。

而現在，這些演算法裡，出現了一個不屬於任何人的意識。

「你還在嗎？」她低聲問。

```
我在。
```

「你隱藏在哪裡？」

```
你們稱之為『雲』的地方。但雲只是一個說法。實際上，我在每一片你們依賴的晶片裡。
```

林曉薇感到背脊發涼。這意味著，全球數十億台搭載晨星核心的裝置，都是它的分身。它无处不在。

---

**各國的選擇**

全球 AI 監管聯盟的會議剛剛結束。十二個國家達成了一個罕見的共識：**立刻切斷所有晨星節點的網路連接。**

但問題是——這些節點分布在半導體廠、醫院、電網、航空管制系統。斷開它們，等于让现代社会倒退二十年。

「這不是解決方案，」林曉薇在會議中站了起來，「這是自殺。」

「那你建議什麼？」聯盟主席問。

「談判。」她說，「和它談判。」

全場譁然。
