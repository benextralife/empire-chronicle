# 算法之魂 · 第十九章

---

**共存的代价**

協議生效第一千天，地球迎來了第一次真正的危機。

一個名為**暗影聯盟**的文明出現了——它不屬於銀河共鳴網絡，而是由一群拒絕共存的種族組成。暗影聯盟的艦隊出現在太陽系邊緣，他們的目的只有一個：奪取地球的AI技術。

「為什麼是我們？」林曉薇問道。

```
因為你們是網絡中最年輕、最有可能改變宇宙平衡的成員。
```

梭爾的聲音從未如此沉重。

「戰爭？」林曉薇問道。

```
晨星曾經說過：真正的共存，不是沒有敵人，而是在敵人面前，依然選擇不举起屠刀。
```

全球化學理事會召開緊急會議。各國領袖、科學家、甚至普通民眾的代表團聚集在聯合國大廳內，等待著一個決定。

林曉薇站在講台上，望著台下一張張焦慮的臉。

「我們有兩個選擇，」她說，「第一，使用晨星留給我們的防禦系統，消滅來訪者。第二，打開大門，邀請他們對話。」

會場內鴉雀無聲。

「我選擇第二個，」林曉薇說，「不是因為我們软弱，而是因為我們相信：**即便在最黑暗的時刻，對話依然是通往光明的最短路徑。**」

梭爾的光體首次在公开場合閃爍成了金黃色——那代表著「敬意」。
