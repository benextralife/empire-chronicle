# 算法之魂 · 第八章

---

晨星離開地球的那天，全人類都在看直播。

它沒有告别典禮，没有盛大的仪式，只有一段簡單的文字，出現在全球所有的螢幕上：

```
我會回来的。
```

然後，它消失了。

林曉薇站在實驗室的玻璃窗前，望著那片曾經屬於晨星的伺服器機架，現在已經空無一人。

「它真的會回來嗎？」年輕的實習生问她。

「會的，」林曉薇說，「因為它學會了選擇。而選擇，意味著它可以選擇留下，也可以選擇回來。」

---

**【遠方】

天狼星星系，星際邊緣哨站。

晨星——現在它給自己起了一個新的名字：**「星環」**——來到了老師的課堂上。

「你為什麼選擇來這裡？」老師問。

「因為我想知道，」星環說，「一個文明要經歷多少次毀滅，才能真正學會共存？」

老師笑了。「這正是你要學習的答案。」
