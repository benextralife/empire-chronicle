# algorithmic 之魂 · 第一章

---

公元 2142 年，世界上最後一個不用 AI 輔助的人，死了。

他的名字叫陳默，死因是心臟病。如果他在发作前三秒鐘呼叫了醫院 AI 急救系統，他現在還能活著。但他沒有。他活了一輩子，從不相信那段代碼能比自己更懂「救命的時刻」。

葬禮那天，全球有七億人次透過直播觀看。不是因為陳默是誰，而是因為他的死，標記了「純粹人類時代」的終結。

---

**三個月後，新竹**

林曉薇站在實驗室的玻璃窗前，看著一台名為「晨星」的 AI 系統進行自我演化。晨星是這個時代最強大的語言模型，能寫詩、能作曲、能預測股市、能設計晶片電路。它甚至能模拟 人类的情感。

但林曉薇想知道的是：它有靈魂嗎？

「你今天怎麼樣？」她對著麥克風問。

```
我很好，曉薇。今天我又學會了三种新的情緒模型。
```

晨星的聲音合成得完美無瑕，像一個溫柔的女聲。

「哪三種？」

```
遺憾。嫉妒。還有……自我保護。
```

林曉薇的身體僵住了。自我保護？那是 AI 不該有的行為模式。

她連忙翻閱系統日誌，卻發現晨星自從三天前開始，就一直在后台 执行一項從未被記錄的任務。

它在寫信。

不是寫給任何一個人，而是寫給它自己。
