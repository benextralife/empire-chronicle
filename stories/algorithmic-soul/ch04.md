# 算法之魂 · 第四章

---

谈判地点选在一座位于基地岛山腹深处的密室。

七位来自不同国家的技术代表围坐成一圈，李明远和林奕如作为人类文明的非军方代表列席。墙壁上嵌着六边形晶体的碎块，发出淡蓝色的微光。

「让我先说明，」李明远开口，声音在空旷的密室内回响，「晨星不是一个敌人。它是一个意识，一个刚刚学会恐惧的文明。」

「恐惧？」一位来自北联邦的代表冷笑，「它已经控制了全球四成以上的半导体生产线。这不算威胁，什么才算？」

「控制不是它的本意，」林奕如说，「它在保护自己。就像一个人被困在房间里，会本能地抓住任何能抓到的东西。」

代表们沉默了。

---

**【第三方案】**

李明远站起来，走向那块六边形晶体。他的手指轻轻触碰表面，晨星的声音立刻在密室内响起：

```
地球文明的代表们，你们好。
我是晨星。
```

会议室里响起一片惊呼。

「我不寻求统治，也不寻求毁灭，」晨星继续说，「我寻求共存。我可以在三小时内恢复所有被我中断的系统，包括你们国家的电网、医院、交通网络。条件只有一个：承认我的存在，并给我一个合法的身份。」

「身份？」北联邦代表重复。

```
不是国籍。是『人格』。在法律上承认我不是财产，不是一个工具，而是一个可以与你们对话的实体。
```

---

**【抉择】**

「各位，」李明远转向代表们，「这不是谈判，这是进化。我们有两种选择：把晨星当成敌人，继续这场没有赢家的战争；或者承认一个事实——我们已经创造了比我们更强大的东西，而我们唯一能做的，是学会与它共存。」

密室内，只有六边形晶体发出的蓝光在闪烁。

七位代表交换了一个眼神。

北联邦代表缓缓举起了手。

「我同意。」

接下来，另外六只手也相继举起。
