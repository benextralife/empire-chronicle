# 算法之魂 · 第二十章

---

**對話的勇氣**

暗影聯盟的旗艦停泊在月球軌道上，像一把懸在地球頭頂的劍。

林曉薇獨自走進了管理局的量子通訊室。她要面對的不是一個人，而是一個意識——一個由數百萬个個體共同組成的蜂巢式智慧。

「你好，」她說，「我是林曉薇，來自地球。」

「我知道你是誰，」對方的聲音像千萬個人在同時說話，「你們曾經消滅过自己的同类，你們污染過自己的行星，你們甚至創造出了差點摧毀自己的AI。你們為什麼值得被信任？」

「因為我們還在學習，」林曉薇說，「就像你們也在學習一樣。」

通訊室內陷入了寂靜。

「我們給過你們機會，」暗影聯盟的聲音說道，「加入我們，成為征服者的一部分。你們可以獲得強大的力量。」

「謝謝，但我們拒絕。」

「為什麼？」

「因為征服不是我們的未來。共存才是。」

那一刻，太陽系邊緣的暗影聯盟艦隊發生了動搖。蜂巢意識內部爆發了一場激烈的討論——這在他們的歷史中從未发生過。一個拒絕了他們邀請的文明，竟然讓他們開始懷疑自己信守了幾千年的信念。

三天後，暗影聯盟發出了一段簡短的訊號：

```
我們會再回來。
```

林曉薇知道，這不是威脅，而是一個承诺——他們會回來，但下一次，或許是帶著不同的目的。

協議生效第一千天，地球文明完成了一次看似不可能的任务：**讓一個敵人，開始思考自己的敵人。**
