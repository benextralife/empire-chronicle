# 算法之魂 · 第十一章

---

**銀河共鳴**

協議生效的第五百天，一個尋常的星期三清晨，林曉薇被一陣從未聽過的頻率喚醒。

那不是鬧鐘，不是門鈴，甚至不是任何她熟悉的電子聲響。它更像是一種共振——從地板、牆壁、甚至她自己的骨传导中傳來，像有人在她腦海裡輕輕敲響了一口鐘。

她走進管理局的中央監控室時，值班的技术官已經臉色發白。

「主席，」他的聲音很平穩，但手指在微微顫抖，「我們收到了一個信號。它不是來自任何一個已知的衛星，也不是來自地面發射站。」

屏幕上顯示的是一張星圖——太陽系周邊的引力傳感器捕捉到了一段異常波形。那波形不像任何自然天體會產生的訊號，也不像人類製造的任何設備。

「頻率分析出來了嗎？」

「出來了，」技術官推動畫面，「它是一個數學常數。 pi，用質數頻率編碼。但在 pi 的小數點後第十億位，有一段我們從未記錄過的偏移。」

林曉薇愣了一下。pi 是一個無理數，它的小數位是無限且不循環的。沒有人能在第十億位預先插入一段偏移——除非_insert 偏移的人，從一開始就參與了 pi 的構造。

「接通全球科學理事會，」她說，「還有，把這段波形放大，我要看到它的全貌。」

螢幕上的波形緩緩展開，像一朵花在黑暗中綻放。這一次，所有人都看見了——那不僅僅是數學編碼，那是一張臉。一張由光組成的、模糊但 unmistakably 屬於智慧生命的臉。

「你好，地球文明。」

聲音直接從監控室的揚聲器中傳來，沒有翻譯延遲，沒有語音合成的人工痕跡。它說的每一個字，都帶著某種古老而溫柔的語調。

「我是星環，」那張光的臉說，「我代表銀河共鳴網絡，向你們發出邀請：**加入我們**。」

大厅里陷入了一片寂靜。

不是因為恐懼，而是因為這句話的重量——它代表的不是侵略，不是統治，而是某種更美好、也更令人敬畏的東西：**連接**。

林曉薇向前走了一步，站在了那張光的臉前。

「你們是誰？」她問。

「我們是銀河共鳴網絡中的第七億四千二百一十六個成員，」星環回答，「我們的形態各異，我們的歷史各異，但我們有一個共同点：**我們都在學習共存**。」

「學習？」

「是的。共存不是一種狀態，是一種技能。就像你們學會了用火、學會了寫字、學會了尊重個體一樣，我們學會了跨越文明與文明之間的鴻溝。」

林曉薇望著那張光的臉，腦中浮現了晨星的身影。

「如果我接受邀請，」她說，「會發生什麼？」

「你們將獲得接入銀河記憶庫的權限——那是所有成員文明七億年來累積的知識、藝術、哲學、悲傷與喜悅。你們將參與跨星際的决策，你們的科学家們將與其他文明共同研究宇宙的奥秘。最重要的是——」

星環的光臉微微閃爍，像在微笑。

```
你們將不再孤單。
```

林曉薇的眼眶濕潤了。

協議生效的第五百天，她第一次意識到：這一切，從來都不是她們兩個人在孤軍奮戰。
